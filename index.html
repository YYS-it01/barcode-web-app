กำลังโหลดสคริปต์...
ถ้าข้อความนี้ไม่เปลี่ยนเป็น “พร้อมใช้งาน” แปลว่าไฟล์ถูกเปิดด้วยตัวพรีวิวที่ไม่รองรับการกดปุ่ม/พิมพ์ข้อความ ให้เปิดด้วย Safari หรือ Chrome แทน
กำลังเริ่มต้นแอพ...

1) โหลดรายการสินค้าอัตโนมัติ

แอพจะดึงข้อมูลจาก Google Sheets อัตโนมัติทันทีตอนเปิดหน้า และสามารถกดโหลดใหม่ได้ทุกเมื่อ
แหล่งข้อมูล-
จำนวนสินค้า0
ระบบจะอ่านข้อมูลผ่าน Google Apps Script ที่เชื่อมกับชีต Products อัตโนมัติ
ข้อกำหนดสำคัญ
ให้ Deploy Google Apps Script เป็น Web App แล้วนำ URL มาใส่ในไฟล์ HTML ตรงตัวแปร GOOGLE_APPS_SCRIPT_WEB_APP_URL เพื่อหลีกเลี่ยงปัญหา CORS

2) ค้นหาและกดเลือกสินค้า

3) สร้างบาร์โค้ด

ระบบใช้ Code-128 ตายตัวทุกครั้ง
แตะรายการสินค้าเพื่อสร้างบาร์โค้ดอัตโนมัติ หรือกรอกเลขเอง
ตัวอย่างบาร์โค้ด
ยังไม่มีข้อมูล

วิธีส่งไฟล์ไปยังโทรศัพท์และวิธีใช้

วิธีส่งไฟล์ไปยังโทรศัพท์
1) กดดาวน์โหลดไฟล์ HTML ลงเครื่องก่อน
2) เปิดไฟล์ด้วย Safari หรือ Chrome เท่านั้น ไม่ใช่หน้าพรีวิวในแชตหรือในแอปส่งไฟล์
3) ต้องมีอินเทอร์เน็ตเพื่อให้แอพโหลดข้อมูลผ่าน Google Apps Script
วิธีใช้บนโทรศัพท์
1) เปิดไฟล์ HTML นี้ใน Safari หรือ Chrome
2) ดูที่หัวข้อ “สถานะการทำงาน” ต้องขึ้นว่า พร้อมใช้งาน
3) รอให้ระบบโหลดข้อมูลจาก Google Apps Script อัตโนมัติจนขึ้นข้อความ “พร้อมใช้งาน”
4) ค้นหาสินค้า แล้วกดรายการสินค้าที่ต้องการ
5) แตะรายการสินค้า ระบบจะสร้างบาร์โค้ด Code-128 ให้ทันที
6) ต้องการรูป ให้กด “ดาวน์โหลด PNG” หรือกด “พิมพ์ / บันทึก PDF”
ถ้ายังพิมพ์หรือกดไม่ได้
แปลว่าตอนนี้ยังเปิดผ่านตัวพรีวิว ไม่ใช่เบราว์เซอร์จริง ให้กด “แชร์” แล้วเลือก เปิดใน Safari หรือ เปิดใน Chrome ก่อน
เคล็ดลับ: Apps Script ควรอ่านชีตที่มีหัวคอลัมน์ รหัสสินค้า, ชื่อสินค้า, หน่วยนับ, บาร์โค้ด และควรตั้งคอลัมน์รหัส/บาร์โค้ดเป็นข้อความเพื่อคงเลขศูนย์นำหน้าไว้
